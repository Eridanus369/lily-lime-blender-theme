LiLy Lime - Blender Theme
A fresh and vibrant Blender theme inspired by LiLy Lime, featuring iconic cyan-green tones paired with pink accents. Optimized for readability and visual appeal, compatible with all Blender versions 3.0+.
Key Features
LiLy Lime Visual Style: Core color scheme based on LiLy Lime's iconic cyan-green (#81d8d0) and cherry blossom pink (#ff69b4), capturing the essence of anime-inspired design.
Full Interface Coverage: Optimized for all core Blender interfaces including 3D View, Node Editor, Properties Panel, and File Browser.
High Readability: Light-toned base with balanced contrast to reduce eye strain during long working sessions.
Detailed Optimization: Customized axis colors, gizmo styles, and node color schemes for both operational recognition and stylistic unity.

Installation Guide
Method 1: Install via Blender Add-ons Menu (Recommended)
Open Blender → Edit → Preferences → Add-ons → Install
Select the downloaded lily_lime_theme.zip file and click "Install Add-on"
Enable the "LiLy Lime" theme in the add-ons list
Method 2: Manual Installation
Download the repository files and extract the lily_lime_theme folder
Copy the folder to the Blender add-ons directory:
Windows: %APPDATA%\Blender Foundation\Blender\3.x\scripts\addons  
Mac: ~/Library/Application Support/Blender/3.x/scripts/addons/
Linux: ~/.config/blender/3.x/scripts/addons/
Open Blender → Edit → Preferences → Themes → Select "LiLy Lime"

Customization Tips
To fine-tune the style, modify these parameters in lily_lime.xml:
Enhance pink contrast: Change text="#ff69b4" to #ff9edf (light cherry blossom pink)
Strengthen cyan-green tone: Change widget_emboss="#7fd6ceff" to #66ccccff (saturated LiLy Lime green)
Adjust grid transparency: Change grid="#137d6d80" to #00cccc40 (lighter cyan-green grid)
